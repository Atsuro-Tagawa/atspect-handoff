# あつぺくとデザイン基準（ATSPECT Design Standard）v1.0

作成：2026-07-22 制作ターミナル（本番テーマ161の実CSS・実表示から抽出した「現に使われている」値のみ。理想論ではなく実測）
用途：営業戦略・リサーチ・システム開発・制作の全ターミナルが、同じ世界観で紙・画面・資料を作るための共通土台。
正本：本ファイル（atspect-handoff/reports/assets/atspect-design-standard.md）。サイト実装が変わったら制作Tが改版する。

---

## 0. 世界観（ひとことで）

**白基調・和紙の温かみ・墨の文字・余白を贅沢に。** 品格と信頼（Professional Credibility）を最優先し、ポップ・煽り・通販的な要素は一切使わない。海外コレクターから見た信頼感を常に意識する。

## 1. 配色（実際に使われているカラーコード）

### 基調色（背景）
| 用途 | カラーコード | 呼び名・説明 |
|---|---|---|
| ページ基調背景 | `#F9F8F6` | 和紙・胡粉色。生成りの極薄温白。ほぼ白だが純白ではない |
| 温かみのある交互セクション背景 | `#f3f0e8` | より温かい生成り。セクションの変化づけに |
| フォーム・申込ゾーンの背景 | `#f4efe6` | 和紙色をやや濃くした落ち着きゾーン |
| ダークセクション | `#0e0e0e` | 使用は限定的（シネマティック演出等） |

### 文字色（墨のグラデーション）
| 用途 | カラーコード | 説明 |
|---|---|---|
| 見出し・本文の主文字 | `#1A1A1A` | 墨色。**純黒 #000 は使わない** |
| 本文の第2主文字 | `#2C2C2C` | やや柔らかい墨 |
| 本文（読み物の長文） | `#3a3a3a` | 長文をやわらげる |
| 補助文字・ヒント | `#595959` / `#6f6a63` | 補足・注記・キャプション |
| ラベル・眉（最小コントラスト基準） | `#6b6b6b` | これより薄い文字は使わない |

### アクセント色（2系統のみ・増やさない）
| 用途 | カラーコード | 説明 |
|---|---|---|
| 金茶（eyebrow・小見出しラベル） | `#a08763` | セクション上の英字ラベル（"Service" 等）に統一使用 |
| ゴールド（ホバー・下線・区切り） | `#b8965a`（ライト `#d4b07a`・薄ティント `rgba(184,150,90,0.12)`） | ナビ下線・アクティブ状態 |
| 群青（行動色＝ボタン・リンク） | `#34416e`（ホバー `#2a3556`） | 申込ボタン・CTA・重要リンク。**緑・オレンジ等の既製品色は使わない** |

### 罫線・区切り
| 用途 | カラーコード |
|---|---|
| 和紙トーンの罫線（フォーム・規約枠・FAQ区切り） | `#E8E2D6` |
| ニュートラルな罫線 | `#e8e8e8` または `rgba(44,44,44,0.08〜0.16)` |

### 禁止事項（配色）
- 純黒 `#000000`・純白背景一面のクリニカルな白は使わない（温白 `#F9F8F6` が基準）。
- 緑・オレンジ・ピンク等の「既製サービス風」の色は使わない（LINEボタンでさえ緑を使わず群青で品位を保っている）。
- 彩度の高い警告色は使わない。エラー表示も深い赤 `#a3322a` / `#B33C3C` で抑える。

## 2. 書体（実際の読み込みフォント）

| 用途 | 欧文 | 和文 | CSS変数 |
|---|---|---|---|
| 見出し | **Cormorant Garamond**（300/400/600・イタリックあり） | **Shippori Mincho**（500/600）→ Noto Serif JP | `var(--font-heading)` |
| 本文 | **Jost**（300/400/500） | **Noto Sans JP**（400/500） | `var(--font-body)` |
| 中国語（簡/繁） | 同上 | Noto Sans SC/TC・Noto Serif SC/TC | `[data-lang]` で自動切替 |
| 韓国語 | 同上 | Noto Sans KR・Noto Serif KR | 同上 |

- 見出しは**セリフ（明朝）**、本文は**サンセリフ（ゴシック）**の組み合わせで統一。
- 本文ウェイトは300〜400の軽め。**太字の多用はしない**（強調は最小限の `<strong>`＝500相当まで）。
- 見出しのウェイトは400が基準（太い700は使わない。細さが品位）。

## 3. 文字サイズ・行間（統一トークン＝実CSS準拠）

| トークン | 値 | 用途 |
|---|---|---|
| 本文 | 16px（長文は17px） | 白地の本文はこれ未満にしない |
| 補助 | 14px（小補助13px） | ヒント・注記 |
| ラベル | 12px（静的ラベル11px） | eyebrow・件数・タグ。**これ未満の文字は禁止** |
| H1 | clamp(28px, 5vw, 40px) | ページ見出し |
| H2 | clamp(22px, 3vw, 28px) | セクション見出し |
| H3 | clamp(18px, 2vw, 20px) | 小見出し |
| 行間（和文） | **1.8**（読み物は2.0前後まで） | 詰めない。ゆったりが基準 |
| 行間（欧文） | 1.7 | |
| 字間 | 見出し 0.05〜0.06em／eyebrow 0.22〜0.3em／本文 0.02〜0.04em | eyebrowは大きく開けて上品に |

## 4. 余白・レイアウトのルール

- **セクション間の縦余白**：基準 200px（小さめの区切りは110px）。窮屈にしない＝余白を贅沢に使うのが最重要ルール。
- **本文の柱幅**：読み物系は最大 720px（本文ブロックは640px）・グリッド系は最大 960〜1200px。行長を読みやすく保つ。
- **文字揃え**：本文は左揃えが基本。ヒーローと本文で揃えを混在させない（1ページ内で揃え方針を統一）。
- **eyebrowパターン**：セクションの頭は「金茶の英字ラベル（letter-spacing大）→ 見出し（明朝）→ 本文」の順で統一。
- **角丸**：ほぼ使わない（0〜2px）。ふっくらした角丸ボタンは世界観に合わない。
- **影**：ほぼ使わない。立体感より罫線と余白で区切る。
- **ボタン**：群青ベタ＋白文字・角丸2px以下・padding大きめ（18px×32〜50px）・ホバーで濃紺または白抜き反転。

---

## 5. 写真・画像のルール（厳守）

- 作品画像・部屋の雰囲気画像に**窓・自然光・太陽光は絶対禁止**。作品の上に影・窓の光・物の影が落ちるのも禁止。**人工照明のみ**。
- 出所不明画像・フリー素材・他者作品の無断使用は禁止。(a) 自製/権利規律に沿う生成、(b) 作家の明確な許諾＋帰属、のいずれかのみ。
- ジャンルアイコン等の線画は stroke `#2C2C2C`・線幅1.6前後の均一細線・ミニマルな一筆書き調。

---

## 6. 言葉のトーン（コピーの基準）

- ✅ 作家への敬意・橋渡し・「日本の美術と世界をつなぐ」・開かれた前向きなトーン。
- ❌ 煽り・急かし・誇張・序列（ランキング的表現）・通販語（「今だけ」「限定」「お得」）。
- ❌ 「有名・無名を問わず」「誰でも」「掲載できる」（表に出さない）。
- 「合格／不合格」は使わない → **「審査通過／お見送り」**に統一。
- 敬称は「作家さま」「お客さま」。

## 7. 使い分け早見（新しく何かを作る人へ）

```
背景        → #F9F8F6（紙・スライドの地色もこれ）
本文        → #1A1A1A・16px・行間1.8・Noto Sans JP / Jost
見出し      → #1A1A1A・明朝（Shippori Mincho / Cormorant Garamond）・ウェイト400
小さなラベル → #a08763（金茶）・大きめの字間・英字大文字
ボタン・リンク → #34416e（群青）
区切り線     → #E8E2D6
余白        → 迷ったら広く
```
